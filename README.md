### Maven
```xml

  <groupId>dev.mini</groupId>
  <artifactId>kakao-i-openbuilder</artifactId>
  <version>1.0.0</version>
</dependency>
```
### Gradle
```groovy
implementation 'dev.mini:kakao-i-openbuilder:1.0.0'
```
